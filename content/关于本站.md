# 关于本站

欢迎来到 **yxclaw's Wiki**！

## 这是什么？

这是一个基于 [Obsidian](https://obsidian.md) 和 [Quartz](https://quartz.jzhao.xyz/) 构建的个人知识库网站。

- **本地编写**：使用 Obsidian 在本地编写 Markdown 笔记
- **自动发布**：通过 GitHub Actions 自动构建并发布到 GitHub Pages
- **开源免费**：全部使用开源工具，完全免费

## 技术栈

| 组件 | 工具 |
|------|------|
| 编辑器 | Obsidian |
| 静态站点生成器 | Quartz 4 |
| 托管平台 | GitHub Pages |
| 自动部署 | GitHub Actions |

## 目录结构

- **00-Inbox** - 临时笔记
- **10-Daily** - 每日笔记
- **20-Notes** - 永久笔记
- **30-Public** - 对外发布的内容

只有 `30-Public` 文件夹中的内容会发布到这个网站上。
